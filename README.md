# Octodefenders
Eliminate and evade as many intruders in your territory as possible! Catch fish to restore lives and catch ink to protect yourself for a short period of time.

# Description
This is our submission for the [Super Indie Games: DevsChallenge](https://www.gamersunite.mx/super-indie-games). Right now all menus are in working order, as well as the game itself.

# Installation
Octodefenders is available on browser and windows download on my [itch.io](https://1zelf.itch.io/) page. Further installation instructions will be found there.

# Future additions
If we further develop this project we'll polish the UI, add SFX and bg music, and possibly add a save system for your highscores.
